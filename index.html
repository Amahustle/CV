Что такое HTML?

HTML (от англ. HyperText Markup Language — «язык гипертекстовой разметки») — язык разметки, определяющий структуру веб-страниц.

HTML - это самый базовый язык веба (Всемирная система публичных веб-страниц в сети интернет. Веб не следует путать с интернетом, сеть лишь использует интернет как среду для передачи информации и данных.)

HTML - не язык программирования, это язык разметки, используемый для определения структуры веб-страниц посещаемых пользователем

Другие технологии, помимо HTML, обычно используются для описания внешнего вида/представления (CSS) или функциональности/поведения (JavaScript) веб-страницы.

Веб страница состоит из 3х основных строительных блоков:

  • HTML
  • CSS
  • JavaScript

Сам по себе HTML это текст в определенном формате. HTML - язык разметки.

Элементы

Документ HTML - это простой текст, поделённый на элементы. Элементы окружены одинаковыми открывающимися и закрывающимися тегами. Каждый тег начинается и заканчивается с угловых скобок (<>). Также есть теги, которые созданы не для добавления текста, например <p>

Следует обратить внимание что концу имени предшествует символ косой черты (слеша) - <p></p>

В пустых элементах закрывающий тег не требуется и не допускается.

Предлагаю рассмотреть HTML элементы подробнее:

HTML элементы <h1> - <h6>

Это элементы заголовков. Всего 6 уровней <h1> наибольший, <h2> наименьший.

Примечания к использованию:

  • Информация о заголовке может использоваться пользовательскими агентами, например, для автоматического создания оглавления для документа
  • Не используйте низкие уровни чтобы снизить размер шрифта: используйте CSS font-size вместо.
  • Избегайте пропуск уровней заголовков: всегда начинайте с <h1>, потом используйте <h2>, и так далее.
  • Вам следует рассмотреть избегание использования <h1> более раза на страницу.

HTML элемент <br />

Делает отступ, устанавливает прервод строки в тексте. Является самозакрывающим элементом.

HTML элемент <hr />

Служит для тематического разделения абзацев, рисует горизонтальную черту.

HTML элемент <a> </a>

Служит для отображения ссылок.

Элементов в HTML большое множество, подробнее ознакомится с ними можно в технической документации.

Атрибуты

Большинство элементов способны к различным вариациям благодаря атрибутам.

Так например, элемент <hr /> имеет следующие атрибуты:

  • align Задаёт правило выравнивания.По умолчанию значение выставлено как left
  • color Задаёт цвет линии
  • noshade Убирает тени
  • size Устанавливает высоту в px
  • width Задаёт длину линии в px либо в %

Пример:

<hr size=29 width=700 color="blue">


Атрибут target="_blank" для элемента <a> дает возможность открывать ссылку в новой вкладке.

Пример:

<a href="https://www.youtube.com/watch?v=DOEtVdkKwcU"target="blank">Курс по HTML</a>
